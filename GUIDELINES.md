# Candidate Evaluation Checklist & Guidelines

## Mục đích

Checklist này giúp đánh giá ứng viên từ Voz.vn một cách nhất quán và xây dựng pipeline tuyển dụng hiệu quả.

---

## 🎯 Phase 1: Initial Screening (Từ Thread/Post)

### Thread-Level Assessment

| Tiêu chí | Weight | Score (1-5) | Notes |
|----------|--------|-------------|-------|
| **Engagement** | ⭐⭐⭐ | | Số replies, views, reactions |
| **Content Quality** | ⭐⭐⭐⭐ | | Technical depth, clarity |
| **Recency** | ⭐⭐⭐ | | Post date, last activity |
| **Community Standing** | ⭐⭐⭐ | | User title, reputation |

**Engagement Scale:**
- 🔥 **Hot**: >100 replies, >10K views
- 📈 **Active**: 50-100 replies, 5K-10K views  
- 📊 **Moderate**: 20-50 replies, 1K-5K views
- 💤 **Cold**: <20 replies, <1K views

### Individual Post Assessment

#### ✅ Green Flags (Tín hiệu tích cực)

- [ ] Mention specific tech stack (Python, K8s, AWS, etc.)
- [ ] Share concrete experience ("3 năm làm X tại Y")
- [ ] Show problem-solving mindset
- [ ] Active in helping others
- [ ] Link to GitHub/portfolio
- [ ] Clear communication style
- [ ] Recent activity (last 30 days)
- [ ] Detailed self-introduction
- [ ] Mention job-seeking status
- [ ] Open to remote/work arrangements

#### ❌ Red Flags (Tín hiệu cảnh báo)

- [ ] Vague statements only
- [ ] No technical specifics
- [ ] Passive/negative attitude
- [ ] Spam-like behavior
- [ ] Very old post (>6 months)
- [ ] One-time poster
- [ ] Unprofessional language
- [ ] No response to questions

## 🔄 Phase 5: Thread Monitoring

### Ongoing Monitoring Checklist

**Weekly Tasks:**
- [ ] Check tracked threads for new replies
- [ ] Scan new threads in CNTT forum
- [ ] Update candidate statuses
- [ ] Follow up with pending contacts

**Monthly Tasks:**
- [ ] Review all "new" status candidates
- [ ] Re-scan high-potential threads
- [ ] Export updated CSV
- [ ] Generate monthly report

### Thread Priority Matrix

| Priority | Criteria | Action |
|----------|----------|--------|
| **P0 - Critical** | Hot thread (>100 replies), active recently | Check daily |
| **P1 - High** | Active thread (50-100 replies), job-related | Check weekly |
| **P2 - Medium** | Moderate engagement, potential candidates | Check bi-weekly |
| **P3 - Low** | Cold thread, archive | Check monthly |

## 🎓 Best Practices

### Do's ✅

1. **Personalize contact messages** - Reference specific posts
2. **Be respectful** - Voz is a community, not a job board
3. **Provide value** - Explain why they might be interested
4. **Follow up appropriately** - Don't spam
5. **Keep detailed notes** - For future reference
6. **Update regularly** - Status and progress
7. **Respect privacy** - Don't share personal info

### Don'ts ❌

1. **Mass messaging** - Generic copy-paste messages
2. **Aggressive recruiting** - Pushy sales tactics
3. **Ignoring context** - Not reading posts properly
4. **Spamming threads** - Don't derail discussions
5. **Ghosting candidates** - Always follow up
6. **Incomplete data** - Fill all relevant fields
7. **Sharing externally** - Keep candidate data private
